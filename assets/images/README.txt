# Imagens Necessárias para a Landing Page "Coluna Viva"

As seguintes imagens são necessárias para completar a landing page:

1. `logo.png` - Logo do produto "Coluna Viva" (recomendado: 180x60px)
2. `favicon.png` - Favicon para a página (recomendado: 32x32px)
3. `product-main.png` - Imagem principal do produto na seção hero (recomendado: 500x500px)
4. `product-offer.png` - Imagem do produto para a seção de oferta (recomendado: 400x400px)
5. `testimonial-1.jpg` - Foto da primeira pessoa para depoimento (recomendado: 400x300px)
6. `testimonial-2.jpg` - Foto da segunda pessoa para depoimento (recomendado: 400x300px)
7. `testimonial-3.jpg` - Foto da terceira pessoa para depoimento (recomendado: 400x300px)
8. `garantia.png` - Selo de garantia de 30 dias (recomendado: 200x200px)
9. `payment-methods.png` - Imagem com os métodos de pagamento aceitos (recomendado: 300x50px)
10. `background.jpg` - Imagem de fundo para seções (opcional)

Nota: Todas as imagens devem ser otimizadas para web para garantir um carregamento rápido da página. Recomenda-se usar formatos como WebP, PNG com transparência para logos e JPG comprimido para fotos.
